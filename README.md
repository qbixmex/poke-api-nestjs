## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# build application
$ yarn build

# production mode
$ yarn start:prod
```

## Build Application

```bash
$ yarn build
```
